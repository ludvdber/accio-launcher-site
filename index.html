Questions fréquentes

C'est légal ?

Le launcher permet de télécharger et installer les jeux, mais vous devez les posséder légalement. Sinon, le téléchargement se fait sous votre responsabilité. Le code est open-source sur GitHub.

C'est gratuit ?

Oui, entièrement. Open-source, pas de pub, pas de tracking.

C'est sûr ?

Code public sur GitHub. HTTPS uniquement, anti-path traversal, prévention Zip Slip. Aucune donnée collectée.

Quelle qualité graphique ?

1920×1080, renderers D3D11 (HP1-HP2) et DGVoodoo2 (HP3-HP8). Antialiasing, SSAO, HDR. Tout est pré-configuré.

Windows 11 ?

Oui. Windows 10 et 11, DirectX 11, 8 Go RAM, 2 Go VRAM.
